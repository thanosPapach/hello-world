npm install pm2 -g

To κάθε ένα από τα δύο node js application έχει ένα package.json

Μέσα στο package.json υπάρχει τμήμα με scripts

Για να τρέξουμε ένα script κάνουμε npm run και το όνομα του scirpt

To pm2 θα πάει να ψάξει πάντα για το start script.

Οπότε αφού κατέβουν τα τελευταία updates και για τα 2 projects με git pull από το repository main του καθενός,
κάνουμε npm install στο directory του καθενός για να εγκατασταθούν οι βιβλιοθήκες.

Μετά πάμε μέσα στο directory του καθενός και για μια φορά τρέχουμε

pm2 start "npm run start" --name="backend"
